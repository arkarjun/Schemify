How this works / why octilinear?

Real geography is messy — wiggly roads, uneven spacing. A schematic map (think subway maps) simplifies geometry, straightens lines, and snaps angles to a small set of directions (usually 45° or 90°) so the result reads cleanly at a glance, at the cost of true geographic accuracy.

This tool: simplifies each line/polygon (Douglas–Peucker), builds a graph of shared points, then iteratively nudges nodes so edges align to the snapped angles while keeping relative topology. At junctions with 3+ connections a perfect fit isn't always possible — this is a fast heuristic, not a full constraint solver, so busy junctions may land a few degrees off-grid.

Open Style & layers to recolor, re-weight, and relabel each layer — or a single segment — once you're happy with the shape.
